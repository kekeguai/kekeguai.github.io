利用OneinStack一键搭建LNMP和Typecho

本文最后更新于：10 天前

首先到下面的网站复制命令，粘贴到Linux里进行安装 时长大概15-30分钟左右
https://oneinstack.com/auto/

环境安装完成后

#进入oneinstack
cd oneinstack
#添加网站
./vhost.sh


虚拟环境配置好后开始进行下一步 安装Typecho
将下面命令依次输入

#进入网站目录
cd /data/wwwroot/xxx.xxxx.xxx
#一键安装命令
wget https://github.com/typecho/typecho/releases/download/v1.1-17.10.30-release/1.1.17.10.30.-release.tar.gz
tar xzf 1.1.17.10.30.-release.tar.gz
cd build
mv * .. 
cd ..
rm -rf 1.1.17.10.30.-release.tar.gz build

要在phpMyAdmin中新创建一个typecho的数据库

之后就可以访问你的网站 进行Typecho的安装
